A plugin to set a custom welcome message and background color.

== Description ==
This plugin allows you to set a custom welcome message and background color for your website's footer.

== Installation ==
1. Upload `custom-welcome-settings` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to 'Welcome Settings' to configure the plugin.

== Frequently Asked Questions ==
= How do I change the welcome message? =
Navigate to 'Welcome Settings' in the admin menu and enter your desired message.

= How do I change the background color? =
Navigate to 'Welcome Settings' in the admin menu and select your desired color.
